## Constitution (Non-Negotiables)

Set this once in `/constitution` before any phase begins. The AI agent will enforce these rules across every spec.

```
1. Multi-tenant: every database query is scoped to the org — no data leaks between tenants.
2. Provider-agnostic AI client: swap Claude / OpenAI / Gemini via ENV variable, zero code changes.
3. Arabic-first outputs: all AI generations support Gulf dialect Arabic + English.
4. All features ship behind feature flags — no broken production experiences.
5. Every endpoint has OpenAPI docs via drf-spectacular at /api/docs/.
6. Cost controls enforced per org — Claude API spend capped at plan limits, hard stop enforced.
```

### `/specify` prompt

```
/specify Build the AI generation engine for the product description plugin.

Input (POST /api/generate/product-content/):
- product_name: string (required)
- category: enum [fashion, food, electronics, beauty, home, other]
- price: decimal (optional, used for tone calibration)
- target_audience: string (optional)
- tone: enum [professional, casual, luxury]
- language: enum [ar, en, bilingual]

Output (JSON):
{
  "title": string,
  "short_description": string (≤ 160 chars),
  "long_description": string (≤ 500 chars),
  "keywords": string[],
  "seo_meta": string (≤ 155 chars)
}

Additional requirements:
- AIClient class with adapter pattern (Claude default, OpenAI/Gemini switchable via AI_PROVIDER env)
- Separate prompt templates for Arabic Gulf dialect and English
- Enforce JSON schema on AI response (retry once on malformed output)
- Track token usage per org: tokens_in, tokens_out, cost_usd, model, created_at
- Redis cache: key = hash(org_id + inputs), TTL = 24 hours
- Per-org monthly token budget middleware: 429 response when limit exceeded
- Streaming SSE endpoint: GET /api/generate/stream/?session_id=X
```

- [ ] Arabic Gulf dialect system prompt template (professional / casual / luxury variants)
- [ ] English system prompt template (professional / casual / luxury variants)
- [ ] `GenerateProductContentView` (DRF APIView) — validates input, calls AIClient, returns structured JSON
- [ ] `AIUsage` Django model: `org, model, tokens_in, tokens_out, cost_usd, created_at`

## Database Models Overview

```python
# Core models across all phases

class Organization(models.Model):
    name = models.CharField(max_length=255)
    clerk_org_id = models.CharField(max_length=255, unique=True)
    plan = models.CharField(choices=['free','starter','pro','enterprise'], default='free')
    monthly_generation_limit = models.IntegerField(default=20)
    generations_used_this_month = models.IntegerField(default=0)
    api_key_public = models.CharField(max_length=64, unique=True)   # pk_xxx
    api_key_secret = models.BinaryField()                            # Fernet encrypted
    stripe_customer_id = models.CharField(max_length=255, blank=True)
    stripe_subscription_id = models.CharField(max_length=255, blank=True)
    created_at = models.DateTimeField(auto_now_add=True)

class AIUsage(models.Model):
    org = models.ForeignKey(Organization, on_delete=models.CASCADE)
    model = models.CharField(max_length=100)  # claude-3-5-sonnet, gpt-4o, etc.
    tokens_in = models.IntegerField()
    tokens_out = models.IntegerField()
    cost_usd = models.DecimalField(max_digits=10, decimal_places=6)
    language = models.CharField(max_length=10)  # ar, en, bilingual
    created_at = models.DateTimeField(auto_now_add=True)

class SallaIntegration(models.Model):
    org = models.OneToOneField(Organization, on_delete=models.CASCADE)
    shop_id = models.CharField(max_length=255)
    access_token = models.BinaryField()  # Fernet encrypted
    token_expires_at = models.DateTimeField(null=True)
    created_at = models.DateTimeField(auto_now_add=True)

class ZidIntegration(models.Model):
    org = models.OneToOneField(Organization, on_delete=models.CASCADE)
    store_id = models.CharField(max_length=255)
    access_token = models.BinaryField()  # Fernet encrypted
    created_at = models.DateTimeField(auto_now_add=True)
```
